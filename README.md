# chatgpt

CLI for ChatGPT

- Please refer to the original source code [here](https://github.com/mattn/chatgpt).

## Usage

```
$ ./chatgpt
```
- Set the environment variable `CHATGPT_API_KEY` to the API key of ChatGPT

## Installation

```
go install github.com/zuizui-kuroiko/chatgpt@latest
```
